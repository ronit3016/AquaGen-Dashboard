# 💧 AquaGen Dashboard

AquaGen is a real-time smart water monitoring dashboard built using Streamlit. It tracks key parameters like flow, water level, and pressure, and detects anomalies instantly to help monitor system performance efficiently.

---

## 🚀 Features
- 📊 Real-time data visualization  
- 🚨 Automatic anomaly detection (flow spikes)  
- 📜 Alert history tracking  
- 📈 Usage trend analysis  
- 📌 Live system metrics  
- 🔄 Auto-refresh dashboard  

---

## 🛠️ Tech Stack
- Python  
- Streamlit  
- Pandas  
- Matplotlib  
- FastAPI  

---


## ⚙️ How to Run

1. Clone the repository
   git clone https://github.com/ronit3016/AquaGen-Dashboard.git

2. Navigate to project folder
   cd AquaGen-Dashboard

3. Install dependencies
   pip install -r requirements.txt

4. Run the app
   Frontend run Frontend.py
